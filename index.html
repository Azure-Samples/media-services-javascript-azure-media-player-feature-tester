Adobe Flash

You can use Microsoft OSMF plugin for the Adobe Open Source Media Framework (OSMF) . We currently support Smooth Streaming and MPEG-DASH content in this player.

Learn More »

iPhone and iPad

You can use the SDK from Microsoft to build rich applications for the iOS platform. We currently support Smooth Streaming and MPEG-DASH content in the SDK.

Learn More »

Android devices

You can use the SDK from Microsoft to build rich applications for the Android platform. We currently support Smooth Streaming and MPEG-DASH content in the player.

Learn More »

Windows devices

You can use the SDK to build rich applications for both Windows 8 and Windows Phone platforms. We currently support Smooth Streaming and MPEG-DASH content in the player.

Learn More »

HTML5

You can use a standard browser that supports HTML5 Video element to playback content. We currently support MP4 or MPEG-DASH content in the player. For MPEG-DASH, we use the dash.js.

Learn More »

Microsoft Silverlight

You can use Silverlight based Player Framework (MMPPF) to playback content from Azure Media Services. We currently support Smooth Streaming and MPEG-DASH content in the player.
