Adobe Flash

You can use the Adobe Open Source Media Framework (OSMF) or Strobe Media Playback (SMP) to playback content from Azure Media Services. Try out the Flash player below

We currently support Smooth Streaming and MPEG-DASH content in the player.

HTML5

You can use a standard browser that supports HTML5 Video element to playback content hosted on Windows Azure Media Services. Try out the HTML5 video player below.

We currently support MP4 or MPEG-DASH content in the player. For MPEG-DASH, we use the dash.js.

View details »

Microsoft Silverlight

You can use Silverlight based Microsoft Media Platform Player Framework (MMPPF) to playback content from Azure Media Services. Try out the Silverlight player below

We currently support Smooth Streaming and MPEG-DASH content with Microsoft PlayReady DRM protection in the player.
